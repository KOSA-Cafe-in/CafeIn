# CAFEIN
카페도 이젠 QR시대!

고객이 단순히 음료를 소비하는 차원을 넘어 편리함·경험·디지털화된 서비스를 기대하며
비대면 주문과 간편 결제에 대한 수요가 크게 증가함에 따라 만들었습니다.

## 주요 기능
- 회원가입 및 소셜로그인
- 게시판 기능
- 스탬프 적립 기능
- 메뉴 추가 및 수정 및 삭제 기능
- 카페 소개글 기능
- 주문 내역 확인 및 처리(비동기화)
- 주문하기 기능
- 결제(포트원)하기 기능
- 장바구니 기능
